# Live Character Counter Tool

A simple Python GUI tool to count **characters, words, and sentences** in real-time as you type.

## Features
- Live updates while typing
- Counts:
  - Characters
  - Words
  - Sentences
- Simple and beginner-friendly

## Requirements
- Python 3.6 or higher
- Tkinter (comes pre-installed with Python)

## How to Run
1. Clone or download this repository
2. Navigate to the project folder in terminal/command prompt
3. Run the script:
   ```bash
   python live_counter.py
